# CURREX

## 소개
P2P 환전 서비스 <br>
: 카메라 인식과 위치 기반 매칭을 통해 집에서 잠자고 있는 외화를 간편하게 환전할 수 있는 P2P 환전 플랫폼

## 기술 스택
- FE: REACT
- BE: Node.js
- DB: MongoDB

## 실행방법
처음 DB 시딩을 위해서 백엔드 디렉토리로 가서 목데이터 시딩을 먼저 진행한다.
그후, Root 디렉토리에서 'npm start' 를 입력한다.
``` cd server<br>
    node mock.js
    cd ..
    npm start
```
